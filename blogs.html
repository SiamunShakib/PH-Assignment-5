Question-1: What are the different ways to select an element in the DOM?

The different ways to select an Element in the DOM are: getElementByID(), getElementsByClassName(), querySelector(), querySelectorAll(), getElementByTagName(), getElementByName().

Question-2: What is the difference between innerHtml, innerText and texContent?

innerHtml: using inner Html we can set the HTML content dynamically inside an element. innerText: By using innerText we can set or see the text of an element. textContent: innerText and textContent are almost similar. But textContent does't check css.

Question-3:What is event delegation in the DOM?

Event delegation means we can handle multiple child element individually by adding a single eventListener to their parent.

Question-4: What is event bubbling in the DOM?

Event bubbling means an event first occurs on the target element and then it spread to its parent element until it reaches the object.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: I can create elements using document.createElement() function. Add: I can add element using append() & appendChild() function. Remove: I can remove an element using remove() & removeChild() function.
